fn pull_internal_progress(&mut self, internal: &mut [CountMap<T>], consumed: &mut [CountMap<T>], produced: &mut [CountMap<T>]) -> bool

Requests changes to the projection of internal work onto each of the scope's outputs, and changes to the number of messages consumed by each of the scope's inputs, and changes to the number of messages producen on each of the scope's outputs. Returns a bool indicating if there is any un-reported work remaining (e.g. work that doesn't project on an output).
